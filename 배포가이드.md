# 배포 가이드 — 스무고개 시험준비 PWA

**한 번 배포해두면**, 학생들에게 링크 하나만 보내면 됩니다. 학생은 링크를 열어 스마트폰에 **홈 화면 앱처럼 설치**할 수 있고, 인터넷이 없어도 한 번 본 화면은 다시 열 수 있습니다.

## 1단계 · 배포 (3분)

### 방법 A — Netlify Drop (가장 쉬움, 추천)

1. https://app.netlify.com/drop 접속
2. 이 **`배포` 폴더 전체**를 브라우저 창에 끌어다 놓기
3. 자동으로 `https://랜덤이름-xxxxx.netlify.app` 주소가 생성됨
4. 상단 **Site settings → Change site name**에서 예쁜 주소로 변경 (예: `smugogae-yeon.netlify.app`)
5. 끝! 이 주소를 학생에게 보내면 됩니다.

### 방법 B — 기존 연프로젝트 서버

서버의 `/smg/` 같은 경로에 이 폴더 내용 전체를 올립니다.
- `index.html`이 루트
- `sw.js`는 **같은 경로**에 있어야 작동(하위 경로면 서비스워커 스코프 제한됨)

### 방법 C — GitHub Pages / Vercel
저장소에 폴더 내용 push → 정적 사이트 설정 → 주소 할당. 동일하게 동작.

## 2단계 · 학생에게 공유

학생 카톡/단체방에 이렇게 보내세요:

```
📚 스무고개 시험 준비
https://<너의_도메인>/

① 위 링크 열기
② "홈 화면에 설치" 버튼 누르기
③ 이제 앱처럼 써!
```

학생 폰 브라우저별 설치 방법:
- **안드로이드 크롬**: 자동으로 "설치" 배너가 뜸 → 한 번만 누르면 끝
- **아이폰 사파리**: 하단 공유 아이콘(⬆︎) → "홈 화면에 추가"
- **아이폰 크롬**: 먼저 사파리로 전환 후 위와 동일

## 3단계 · 선생님 쪽 동작 확인

학생이 학습을 마치면:

1. 학생 앱에서 **난이도·이해도·부족한 부분** 입력 후 "기록 저장하기"
2. "선생님께 전달하기" 섹션이 열림
3. 학생이 다음 중 하나를 선택:
   - **JSON 복사** → 카톡으로 보냄
   - **링크 보내기** → 카톡/공유 시트로 보냄
   - **카톡 메시지로 보내기** → 요약 텍스트 복사됨
   - **6자리 코드** → 수업 중 구두로 알려줌
4. 선생님은 받은 링크를 누르면 `teacher-view.html`이 열려 **학생 이력이 표로 펼쳐짐**
   - 평균 난이도/이해도, 과목 수, 기록 수 자동 집계
   - JSON 복사 → 연플래너에 붙여넣기
   - 인쇄도 지원

## 4단계 · 연플래너 연동 (다음 단계)

아직 자동 연동은 없습니다. 현재 흐름:

```
학생 앱 → (JSON/링크/코드) → 선생님 → 연플래너 수동 입력
```

다음 마일스톤:
- **2차 연동**: 연플래너에 "시험 학습 분석" 화면 추가 → JSON 붙여넣기만 하면 자동 파싱
- **3차 연동**: 본부 서버 API 연결 → 학생이 버튼 한 번으로 자동 전송

연동 포맷은 **DATA-SCHEMA.md** 문서 참조.

## 5단계 · 업데이트 방법

HTML을 수정한 뒤:

1. 파일을 배포 폴더에 덮어쓰기
2. **`sw.js`의 `CACHE_VERSION`을 `smg-v2` 등으로 증가** (매우 중요)
3. Netlify 재배포(Drop에 다시 끌어놓기)

학생 폰에서는 다음 앱 실행 시 자동으로 새 버전 받아옵니다 (service worker가 차이 감지).

## 6단계 · 점검 체크리스트

- [ ] `index.html`이 브라우저에서 열리는가?
- [ ] 아이콘이 제대로 보이는가?
- [ ] "설치" 배너가 안드로이드에서 뜨는가? (iOS는 가이드 안내만)
- [ ] `student.html`에서 기록 저장 → 링크 공유가 동작하는가?
- [ ] 공유 링크를 다른 기기/브라우저에서 열면 `teacher-view.html`이 학생 기록을 표시하는가?
- [ ] 오프라인 모드로 전환 후에도 `index.html`이 열리는가? (PWA 캐싱 확인)

## 파일 구성

```
배포/
├── index.html              ← 랜딩 페이지 (PWA 시작점)
├── student.html            ← 스무고개 학생 앱 (기록 저장/전달 포함)
├── photo-editor.html       ← 교과서 사진 보정 도구
├── photo-guide.html        ← 교과서 촬영 가이드
├── teacher-view.html       ← 선생님 수신용 뷰어
├── manifest.json           ← PWA 매니페스트
├── sw.js                   ← 서비스 워커 (오프라인 + 캐싱)
├── icon-192.png            ← PWA 아이콘 (Android)
├── icon-512.png            ← PWA 아이콘 (고해상도)
├── icon-maskable.png       ← Android 적응형 아이콘
├── apple-touch-icon.png    ← iOS 홈 화면 아이콘
├── favicon.png             ← 브라우저 탭 아이콘
├── DATA-SCHEMA.md          ← JSON 스키마 명세
└── 배포가이드.md            ← 이 파일
```
